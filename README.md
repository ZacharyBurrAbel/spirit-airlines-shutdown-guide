# Spirit Airlines Shutdown — Passenger Rights Guide

A clean, mobile-responsive dashboard helping Spirit Airlines passengers understand their rights and options after the airline's May 2, 2026 shutdown.

## What's in this guide

- **Refund situation by payment type** — credit card, debit card, third-party booking, points/vouchers
- **Rescue fares** from American, United, Delta, JetBlue, Southwest, Frontier with deadlines
- **Step-by-step action checklist** in priority order
- **Key contacts** — Epiq claims agent, JetBlue booking line, DOT complaints

## How to use on GitHub Pages

1. Fork or clone this repository
2. Go to your repo **Settings → Pages**
3. Under "Source," select **Deploy from a branch**
4. Choose `main` branch and `/ (root)` folder
5. Click Save — your site will be live at `https://yourusername.github.io/your-repo-name/`

## How to embed in WordPress (iframe)

Once your GitHub Pages site is live, add this to any WordPress page or post using the **Custom HTML** block:

```html
<iframe
  src="https://yourusername.github.io/your-repo-name/"
  width="100%"
  height="2800"
  frameborder="0"
  scrolling="auto"
  style="border: none; border-radius: 12px;"
  title="Spirit Airlines Passenger Rights Guide">
</iframe>
```

**Tips for WordPress:**
- Use the **Custom HTML** block (not the paragraph block) to paste iframe code
- Adjust the `height` value if content gets cut off (try 2800–3200px)
- For full-width display, make sure your WordPress page uses a full-width template
